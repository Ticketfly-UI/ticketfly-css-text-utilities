_Ticketfly CSS helpers for the "tap-target" object pattern_.


## Installation

Installing with `npm`:

```bash
npm install ticketfly-css-text-utilities
```

Installing with `yarn`:

```bash
yarn add ticketfly-css-text-utilities
```
